# Personal Portfolio Website

This is my personal portfolio website built to showcase my skills, projects, and resume.

The website is designed with a clean and minimal UI and highlights my learning journey in web development.

## Features
- Responsive and modern design
- About Me and Skills section
- Projects showcase
- Resume download option
- Deployed live using GitHub Pages

## Technologies Used
- HTML
- CSS
- JavaScript

## Live Website
https://cherryuserhub.github.io/personal-website/

## Purpose
This project was created to practice frontend development and to build a professional online presence for internships, placements, and learning purposes.
